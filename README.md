# Результат разработанного тестового задания

Задача заключалась в разработке небольшого личного кабинета с использованием JWT и
возможностью редактирования информации авторизованного пользователя

### Для запуска сервера
1. cd server
2. npm install
3. npm run dev

### Для запуска клиента
1. cd client
2. npm install
3. npm start
